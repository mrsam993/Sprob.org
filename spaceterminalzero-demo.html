Space Terminal Zero Demo

As mentioned in the learn more page this game is not totally finished and serves more as a proof of concept than a fully functioning game, so if you have not already I would reccomend reading the learn more page to understand what is going on. Despite not being finished I am very proud of what I did manage to make under the time constraints of a GameJam. The controls are randomized each game and can be read in the logbook which is read using the arrow keys. (Also yes I do know there is a typo on the main menu we all make mistake when under time pressure).
